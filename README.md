![VideoScreensaver](icon.png)

VideoScreensaver is a screensaver that will play a video file when the screensaver starts. It will support playing either a user specified video file, or one of the following pre-defined video files detailed on the Wiki.

[Add-on:VideoScreensaver](http://kodi.wiki/view/Add-on:VideoScreensaver)

You can install this addon from: [Repository:robwebset](https://github.com/robwebset/repository.robwebset/blob/master/repos/repository.robwebset/repository.robwebset-1.0.0.zip) ([wiki](http://kodi.wiki/view/Repository:robwebset))
